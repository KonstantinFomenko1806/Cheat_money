Данная программа взаимодействует с данными, загруженными в оперативную память.

1) Умеет искать адрес ячеек памяти с нужными для нас данными. Можно произвести несколько итераций поиска с вводом нового значения. 
   В конце выведет актуальные адреса ячеек памяти, значения которых совпадали на каждой итерации поиска.

2) Умеет читать значение по введенному пользователем адресу памяти. (в основном для проверки записанного значения)

3) Умеет записывать введенное пользователем значение по адресу в память.
